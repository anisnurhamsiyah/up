Menambahkan Gambar di HTML

Gambar dapat kita tambakan di HTML dengan menggunakan tag img. Tag ini memiliki atribut wajib, yakni src, contoh:

Jika kita tidak mengisi atribut src, maka gambar tidak akan ditampilkan. Alamat URL gambar pada atribut src dapat berupa URL maupun alamat path. Lalu tag img harus ditutup dengan menambahkan garis miring. Menuliskan langsung nama file dari gambar karena gambar berada di dalam folder yang sama dengan file HTML. Apabila file gambarnya tersimpan di folder yang berbeda, maka kita hanya perlu menuliskan alamat path-nya. Jika kita menggunakan gambar dari website lain kita harus menuliskan alamat URL lengkap dari gambar tersebut.

Atribut untuk Tag img

Ada beberapa atribut yang sering digunakan pada tag img:
  • alt untuk teks alternatif untuk gambar;
  • width untuk menentukan lebar gambar;
  • height untuk menentukan tinggi gambar;
  • style untuk menentukan style CSS untuk gambar.
  • title untuk menentukan nama dari gambar

Membuat Gambar Background

Agar gambar bisa ditampilkan sebagai background, maka kita harus menggunakan CSS dengan atribut background-image lalu diisi dengan URL dari gambar. Atribut tersebut diisi pada tag body.
